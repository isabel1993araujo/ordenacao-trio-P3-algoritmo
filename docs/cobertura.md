# Relatório de Cobertura (gcov)

A ferramenta gcov foi utilizada para medir a cobertura de código dos algoritmos implementados.

## Resultados

| Arquivo | Cobertura |
|---------|-----------|
| test_basic.c | 100% |
| bubble.c | 100% |
| selection.c | 100% |
| insertion.c | 100% |
| merge.c | 100% |
| quick.c | 100% |
| heap.c | 100% |

## Análise

Todos os algoritmos apresentaram cobertura de 100%, indicando que todas as linhas relevantes foram executadas durante os testes.

Os testes foram realizados com:
- vetores aleatórios
- vetores invertidos

Isso garantiu a validação do comportamento dos algoritmos em diferentes cenários.
