**Disciplina:** Algoritmos e Estruturas de Dados

**Avaliação:** P3  
**Linguagem:** C (padrão C99 ou C11)

## Descrição

Este projeto implementa algoritmos de ordenação em linguagem C, incluindo algoritmos básicos e avançados, com foco em análise de desempenho, organização de código e uso de Git/GitHub.

## Algoritmos implementados

### Básicos
- Bubble Sort
- Selection Sort
- Insertion Sort

### Avançados
- Merge Sort
- Quick Sort
- Heap Sort
